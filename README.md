# 팬레터 개인과제

좋아하는 아티스트 그룹이 없어서 캠프내에서 판교를 가기위한 스터디 하는 그룹원들을 대상으로 작성함.

## 주요 기능

- 각 인원별 응원 메세지 작성
- 메세지 작성시 select로 인원 선택시 헤더에 인원 버튼 이동(선택한 인원이 누군지 확실히 알기위해 적용)
- 디테일 페이지에서 수정 삭제 기능
